## Getting Started

If you're new to Paper plugin development:
1. Review our [Getting Started Guide](/paper/dev/getting-started)
2. Check out the [Example Plugins](/paper/dev/examples)
3. Join the [PaperMC Discord](https://discord.gg/papermc) for developer support

:::tip[Best Practices]
- Always use Paper's enhanced APIs when available
- Follow the plugin lifecycle guidelines
- Implement proper error handling
- Use the latest Paper API features for best performance
:::

Make sure to check the [Paper Roadmap](/paper/dev/api/roadmap) for upcoming API changes and version compatibility information.
